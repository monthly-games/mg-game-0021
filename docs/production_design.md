## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **오염 제거** | 퍼즐을 통한 오염물 정화의 쾌감 | ★★★★★ |
| **지구 복원** | 정화 후 자연 생태계 복원 시각화 | ★★★★★ |
| **사회적 기여** | 실제 환경 보호 캠페인 연계 | ★★★★☆ |
| **클리너 수집** | 다양한 능력의 클리너 영웅 육성 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **정화 완료**: 오염된 지역을 완전히 정화했을 때의 성취감
- **생태계 복원**: 동식물이 돌아오는 복원 애니메이션
- **실제 기여**: 게임 활동이 실제 기부로 연결되는 의미
- **캠페인 클리어**: 브랜드 콜라보 이벤트 완료 보상

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 오염 유형별 퍼즐 전략
├── 디펜스 웨이브 방어
├── 제한 시간 내 정화
├── 클리너 조합 최적화
└── 캠페인 목표 달성
```

### 1.4 사회적 요소 (Social)
- **글로벌 캠페인**: 전체 플레이어 누적 정화량
- **지역 랭킹**: 지역별 환경 기여도 경쟁
- **소셜 챌린지**: 친구와 함께하는 정화 미션
- **실제 연동**: 환경 단체 기부 참여

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (3~5분) - 스테이지 플레이
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [클리너 배치] → [오염 퍼즐] → [디펜스]           │
│       ↑                            ↓             │
│       └──── [보상] ← [지역 정화] ←┘              │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **클리너 배치** (15초): 스테이지에 맞는 클리너 선택
2. **오염 퍼즐** (2~3분): 매치3/연결 퍼즐로 오염 제거
3. **디펜스** (1~2분): 밀려오는 오염 공격 방어
4. **지역 정화** (즉시): 목표 달성 시 정화 완료
5. **보상** (10초): 에코 포인트, 복원 진행

### 2.2 세션 루프 (10~20분)
```
[접속] → [일일 미션] → [스테이지 진행] → [지역 복원] → [클리너 강화]
                                              ↓
                              [캠페인 참여] ← [에코 포인트 수집]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [스테이지 클리어] → [지역 복원] → [새 지역 해금]     │
│         ↓                              ↓             │
│  [클리너 수집]                   [브랜드 캠페인]      │
│         ↓                              ↓             │
│  [클리너 강화] ←──────── [임팩트 토큰 획득]          │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 화면] ─────→ [클리너 관리]
    │              [지역 맵]
    │              [캠페인]
    │              [설정]
    ↓
[스테이지] → [클리너 선택] → [퍼즐] → [디펜스] → [결과]
    │
    └── [캠페인 이벤트] → [특별 스테이지] → [브랜드 보상]
```

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
난이도
  ▲
  │                              ╭── 방사능 오염
  │                         ╭────╯ 토양 오염
  │                    ╭────╯ 수질 오염
  │               ╭────╯ 대기 오염
  │          ╭────╯ 플라스틱 오염
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 지역
    해변   도시   공장   농촌   핵시설
```

**난이도 공식:**
```
오염_강도 = 기본_강도 × 지역_배율 × (1 + 스테이지 × 0.15)
퍼즐_복잡도 = 오염_종류 × 조합_요소
```

### 4.2 콘텐츠 진행
| 지역 | 오염 유형 | 새로운 요소 |
|------|----------|-------------|
| 해변 | 플라스틱 | 기본 퍼즐, 튜토리얼 |
| 도시 | 대기 | 디펜스 모드, 콤보 |
| 공장 | 수질 | 특수 오염, 보스 웨이브 |
| 농촌 | 토양 | 복합 오염, 클리너 시너지 |
| 핵시설 | 방사능 | 극한 난이도, 특수 클리너 |

### 4.3 밸런싱 공식
```dart
// 정화 점수
int calculatePurificationScore(Stage stage, Performance perf) {
  int baseScore = stage.pollutionLevel * 10;
  double timeBonus = max(0, 1 - perf.time / stage.timeLimit) * 0.3;
  double comboBonus = perf.maxCombo * 0.1;
  return (baseScore * (1 + timeBonus + comboBonus)).toInt();
}

// 클리너 효과
double cleanerEffect(Cleaner cleaner, PollutionType type) {
  double base = cleaner.power;
  double typeBonus = cleaner.specialties.contains(type) ? 1.5 : 1.0;
  double levelBonus = 1 + cleaner.level * 0.05;
  return base * typeBonus * levelBonus;
}

// 복원 진행도
double restorationProgress(Region region) {
  int totalStages = region.stages.length;
  int cleared = region.clearedStages;
  return cleared / totalStages;
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 에코 포인트 | 스테이지 클리어 | 난이도별 50~300 |
| 그린 젬 | 지역 완료 | 10~50 |
| 클리너 조각 | 보스 격파 | 1~5 |
| 임팩트 토큰 | 캠페인 참여 | 이벤트별 |
| 복원 보상 | 생태계 완성 | 특별 보상 박스 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 에코 포인트/그린 젬
├── 에너지
├── 복원 진행률
└── 캠페인 배너

레벨 2 (상호작용 시):
├── 클리너 상세
├── 오염 정보
├── 퍼즐 힌트
└── 지역 상태

레벨 3 (팝업):
├── 스테이지 결과
├── 복원 연출
├── 클리너 획득
└── 캠페인 상세
```

### 5.2 네비게이션 흐름
```
[메인 화면]
    ├── [지역 맵] → [지역 선택] → [스테이지] → [플레이]
    ├── [클리너] → [클리너 목록] → [강화/진화]
    ├── [캠페인] → [이벤트 목록] → [참여]
    ├── [상점] → [그린 젬/팩]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 오염 제거 | 정화 이펙트 | 청소 효과음 | 진동 (약) |
| 콤보 | 콤보 연출 | 콤보 사운드 | - |
| 지역 정화 | 자연 복원 연출 | 자연 소리 | 진동 (중) |
| 생태계 완성 | 동물 귀환 연출 | 팡파레 | 진동 (강) |
| 캠페인 완료 | 브랜드 연출 | 성취음 | 진동 (중) |

### 5.4 접근성
- **간단한 조작**: 탭과 스와이프 기반
- **색맹 모드**: 오염 유형별 패턴 추가
- **폰트 크기**: 3단계 조절
- **환경 정보**: 교육 콘텐츠 접근 용이
- **AR 모드**: 실제 환경 스캔 (옵션)
