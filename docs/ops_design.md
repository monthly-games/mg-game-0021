## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 청소 미션 밸런스 |
| 7~10주 | 첫 환경 캠페인 준비 |
| 11~12주 | 첫 브랜드 콜라보 협의 |

**핵심 과제:**
- 청소 게임플레이 재미
- 환경 메시지 전달
- 브랜드 콜라보 기반 마련

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 해양 플라스틱 청소 캠페인 |
| 5개월 | 첫 브랜드 콜라보 실행 |
| 6개월 | 숲 복원 캠페인 |
| 7개월 | 브랜드 콜라보 2차 |
| 8개월 | 멸종위기종 구출 캠페인 |

**핵심 과제:**
- 시즌별 환경 캠페인 사이클
- 브랜드 콜라보 효과 검증
- 기관 파트너십 확대

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 기관 인증 연계 |
| 10개월 | 실제 기부 연동 테스트 |
| 11개월 | 대규모 환경 캠페인 |
| 12개월 | 1주년 그린 페스티벌 |

**핵심 과제:**
- 기관 파트너십 강화
- 실제 행동 연계 가능성
- 장기 유저 리텐션
