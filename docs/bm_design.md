## 3. 상세 BM 설계

### 3.1 스태미너/편의 BM

#### 무료 구조
- 스태미너 최대 50, 5분당 1 회복
- 일일 무료 충전 1회
- 미션 클리어 보상으로 스태미너 획득

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 스태미너 즉시 충전 | 50 회복 | 프리미엄 재화 80 |
| 스태미너 패스 | 최대 +20, 회복 3분 | $4.99/월 |
| 광고로 스태미너 | 10 회복 | 무료 (광고) |

### 3.2 클리너/장비 BM

#### 무료 구조
- 스토리 진행으로 기본 클리너 3종 합류
- 미션 완료로 장비 획득
- 업적 달성으로 특수 클리너 해금

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 클리너 해금 | 개별 클리너 구매 | $2.99~$4.99 |
| 클리너 번들 | 클리너 3종 + 장비 | $9.99 |
| 클리너 스킨 | 유니폼/외형 변경 | $1.99~$3.99 |
| 프리미엄 장비 | 고급 청소 장비 | $2.99~$4.99 |

### 3.3 꾸미기/테마 BM

#### 무료 구조
- 미션 클리어로 기본 장식 획득
- 복원된 환경 자동 해금
- 이벤트에서 한정 데코 파밍

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 환경 테마 세트 | $4.99~$9.99 |
| 복원 장식 | 특수 환경 장식 | $1.99~$3.99 |
| 팀 하우스 | 본부 꾸미기 | $2.99~$4.99 |

### 3.4 브랜드 콜라보 BM

| 콜라보 유형 | 내용 | 가격대 |
|------------|------|--------|
| 브랜드 미션 | 스폰서 미션 + 보상 | 무료/유료 |
| 브랜드 장비 | 콜라보 한정 아이템 | $4.99~$9.99 |
| 기관 인증 | 환경 단체 인증 마크 | 기부 연동 |

### 3.5 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 스태미너, 청소 효율 UP, 보상 2배 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 미션 무과금 클리어 가능
- 클리너/장비 성능 차이 최소화
- 테마/메시지 중심 설계

### 과금 압박 최소화
- 캐주얼 유저는 스태미너 부족 느끼지 않는 수준
- 친환경 메시지와 맞는 부담 없는 과금 구조
- 브랜드 콜라보로 수익 다각화
