## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `energy_refill` | 에너지 부족 | 에너지 +5 | 5회 |
| `cleaner_boost` | 스테이지 시작 | 정화력 +20% | 5회 |
| `reward_double` | 스테이지 클리어 | 보상 2배 | 5회 |
| `revive` | 정화 실패 | 즉시 재도전 | 3회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |

### 1.2 전면 광고

- 스테이지 완료 후 (3스테이지마다)
- 쿨다운 3분
- 구독자 제외

## 6. 공정성 원칙

- 모든 스테이지 무과금 클리어 가능
- 실력 기반 진행 (Pay-to-Win 금지)
- SSR 클리너는 장기 플레이로 획득 가능
- 스킨/테마는 게임플레이에 영향 없음
- 가챠 천장: 60회 SSR 확정
