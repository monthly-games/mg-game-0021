## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 오염 지역 탐지/진입                                  │
│         ↓                                               │
│  2. 퍼즐/디펜스로 오염 제거                              │
│         ↓                                               │
│  3. 청소율에 따른 보상 획득                              │
│         ↓                                               │
│  4. 장비/팀 업그레이드 ─────────────────→ (1로 반복)      │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **오염 탐지**: 오염된 지역 발견 및 미션 시작
2. **오염 제거**: 퍼즐 또는 디펜스 방식으로 오염 청소
3. **보상 획득**: 청소율/효율에 따른 보상
4. **업그레이드**: 청소 장비, 클리너 팀 강화

## 3. 재미 축 (Fun Pillars)

### 3.1 퍼즐/디펜스 (Puzzle/Defense)
- **목표**: 전략적 사고가 필요한 게임플레이
- **핵심 경험**: 효율적으로 청소 완료했을 때의 성취감
- **주요 요소**:
  - 매치 퍼즐 청소
  - 타워 디펜스 모드
  - 복합 미션
  - 보스 오염원

### 3.2 테마 몰입 (Theme)
- **목표**: 환경 보호 메시지를 재미있게 전달
- **핵심 경험**: 내가 환경을 지키고 있다는 의미 부여
- **주요 요소**:
  - 환경 복원 비주얼
  - 생태계 회복 스토리
  - 실제 환경 이슈 연계
  - 교육적 콘텐츠

### 3.3 미션/목표 (Mission)
- **목표**: 명확한 목표 달성의 재미
- **핵심 경험**: 미션 완료! 하는 달성감
- **주요 요소**:
  - 일일/주간 미션
  - 챕터별 목표
  - 특별 미션
  - 업적 시스템

## 4. 디자인 레버 (Design Levers)

### 4.1 시각적 오염도 표현
| 레버 | 설명 |
|------|------|
| 직관적 시각 효과 | 오염도/청소율을 직관적으로 보여주는 시각 효과 |
| 구현 방향 | 오염 전/후 대비 강조, 청소 진행 바, 생태계 복원 애니메이션 |

### 4.2 테마 퀘스트
| 레버 | 설명 |
|------|------|
| 친환경 메시지 재미화 | 친환경 메시지를 재미로 소화할 수 있는 퀘스트/이벤트 설계 |
| 구현 방향 | 해양 플라스틱 청소 이벤트, 숲 복원 캠페인, 멸종위기종 구출 |

### 4.3 콜라보 구조
| 레버 | 설명 |
|------|------|
| 미션 구조 확장 | 브랜드/기관 콜라보를 끼워 넣을 수 있는 미션 구조 확보 |
| 구현 방향 | 스폰서 미션 슬롯, 브랜드 아이템 보상, 기관 인증 마크 |
